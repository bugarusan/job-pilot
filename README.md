# JobPilot

AI-powered job assistant designed to help users find suitable job opportunities, optimize their CVs, and manage job applications efficiently.

---

# 🇬🇧 English

## About

JobPilot is an AI-powered job assistant that helps users:

- Find suitable job opportunities
- Track new job postings
- Optimize and rewrite CVs based on job requirements
- Match skills and experience with job descriptions
- Manage job applications in one place

## Planned Features

- Job aggregation from multiple platforms
- AI-powered CV analysis
- Automatic CV tailoring for each job posting
- Job matching system
- User profiles
- Application tracking dashboard
- Notifications for new relevant jobs

## Technology Stack

- Java 21
- Spring Boot
- PostgreSQL
- Maven
- Git & GitHub

## Status

🚧 Currently under development.

# 🇯🇵 日本語

## 概要

JobPilot は、ユーザーが自分に適した求人を見つけ、履歴書（CV）を改善し、応募状況を管理できる AI 搭載のジョブアシスタントです。

## 予定している機能

- 複数の求人サイトから求人情報を収集
- AI による履歴書分析
- 求人内容に合わせた履歴書の自動最適化
- 求人マッチングシステム
- ユーザープロファイル
- 応募状況管理ダッシュボード
- 新しい求人通知

## 使用技術

- Java 21
- Spring Boot
- PostgreSQL
- Maven
- Git & GitHub

## 開発状況

🚧 現在開発中です。
